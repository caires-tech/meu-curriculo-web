# Currículo - Rodrigo Caires - Profissional Administrativo & Tecnologia

Bem-vindo ao meu currículo profissional! 
Este projeto foi desenvolvido com o objetivo de centralizar minha trajetória e destacar minhas competências técnicas e administrativas.

## Sobre o Projeto
Este site apresenta uma visão consolidada da minha carreira, unindo 23 anos de experiência operacional com minhas novas competências em desenvolvimento de software.

## Tecnologias Utilizadas
- **HTML5**: Estrutura semântica e organizada.
- **CSS3**: Design responsivo, estilização personalizada e alinhamento de componentes.
- **JS**: Controle de abertura e fechamento dos cards.
- **Git/GitHub**: Controle de versão e publicação via GitHub Pages.
